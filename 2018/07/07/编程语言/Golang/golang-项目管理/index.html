Go vendor

要求 go 版本 大于 1.6

注意事项：

  • vendor 只能在 package 中存在，不能超越这个范围。也就意味着 vendor 不能是 $GOPATH/src 一级目录，否则报错。
  • vendor 可以嵌套使用，就是 vendor 里面可以还有 vendor

好处：

  • 多项目管理，有自己的包依赖，工程隔离
  • 可以平滑从非 vendor 项目转换为 vendor 项目
  • vendor 可以记录项目依赖包及其版本信息，任意升级换代，也任意维护

一个流程

  • cd /path/to/your/project
  • 写好之后 dep init -v
  • 运行：go run main.go
  • 编译：go build -o www
